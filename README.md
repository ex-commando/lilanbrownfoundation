# Lilian Brown Foundation Website

A modern, high-fidelity React application for the **Lilian Brown Foundation (RC: 9197336)**, an NGO dedicated to empowering women, girls, and vulnerable groups through education, advocacy, and sustainable development.

## 🌟 Features

- **Dynamic Hero Slider**: A polished background image carousel with centralized content overlays and smooth transitions.
- **Multi-Page Architecture**: Developed with `react-router-dom` for a structured user experience across:
  - **Home**: Overview of mission, objectives, and transparency.
  - **About Us**: Detailed mission statement and strategic pillars.
  - **Governance**: Comprehensive structural details of the Board of Trustees and meetings.
  - **Contact**: Leadership profiles, headquarters details, and registration info.
  - **Transparency**: Dedicated area for financial accountability.
- **Premium Aesthetics**: Glassmorphism effects, a custom Pink & Grey color palette (from the logo), and high-quality representative imagery.
- **Fully Responsive**: Mobile-first design with a dedicated sidebar menu for smaller screens.

## 🛠️ Tech Stack

- **Core**: React.js 19
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (CSS Variables + Flexbox/Grid)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm (Node Package Manager)

### Installation

1. Clone or download the project repository.
2. Navigate to the project directory:
   ```bash
   cd Lilianbrownfoundation
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173/`.
